## Requisiti
Scritto in [python (3.7+)](https://www.python.org/downloads/) sfruttando le librerie [discord.py](https://github.com/Rapptz/discord.py) e dotenv.
Per installarle:
```
python3 -m pip install -U discord.py

python3 -m pip install -U dotenv
```
